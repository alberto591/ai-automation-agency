# Request for Proposal (RFP): Historical Real Estate Transaction Data

**To**: Sales Department (Nomisma / Scenari Immobiliari)
**From**: Anzevino AI Team
**Subject**: Request for Bulk Historical Real Estate Transaction Data for AVM Training

## 1. Project Overview
Anzevino AI is developing "Fifi", a production-grade Automated Valuation Model (AVM) for the Italian residential market. We are seeking high-quality, verified historical transaction data to train and validate our machine learning models.

## 2. Requirements
We are looking for a bulk dataset covering the last 24-36 months for the following cities: **Milan, Rome, Florence, Turin, and Naples**.

### 2.1 Data Points Required (per transaction)
- **Exact Address & Postcode** (or Micro-zone/Census tract)
- **Final Sale Price** (Price from Notarial Deed, not asking price)
- **Transaction Date**
- **Property Size (sqm)**
- **Building Characteristics**: Floor, Elevator, Year of Construction, Balcony/Terrace.
- **State of Repair**: (Excellent/Good/Fair/Poor).
- **Cadastral Category**: (A/1, A/2, A/3, etc.)

### 2.2 Volume & Format
- **Target Volume**: Minimum 10,000 records per city.
- **Format**: CSV, Parquet, or SQL dump.
- **Frequency**: One-time bulk purchase with potential for quarterly updates.

## 3. Inquiry Points
1.  **Coverage**: Can you provide data samples for the Milan/Rome metropolitan areas?
2.  **Accuracy**: What is the source of the price data (e.g., Agenzia delle Entrate, Notarial Archives)?
3.  **Licensing**: What are the terms for using this data to train a proprietary ML model for commercial use?
4.  **Pricing**: Please provide a tiered quote based on the number of records or cities covered.
